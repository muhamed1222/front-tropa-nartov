# Инструкция по включению клавиатуры в iOS симуляторе

## Способ 1: Через меню симулятора (самый простой)

1. Откройте iOS симулятор
2. Перейдите в меню: **I/O → Keyboard**
3. Убедитесь, что **"Connect Hardware Keyboard"** НЕ отмечен (снимите галочку)
4. Или используйте горячую клавишу: **⌘K** (Command + K)

После этого клавиатура будет появляться на экране симулятора при нажатии на поля ввода.

## Способ 2: Через настройки симулятора

1. В симуляторе откройте **Settings** (Настройки)
2. Перейдите в **General → Keyboard**
3. Убедитесь, что все настройки клавиатуры включены

## Способ 3: Горячая клавиша

Нажмите **⌘K** (Command + K) для быстрого переключения режима клавиатуры.

## Проверка в коде

В коде приложения уже настроено автоматическое открытие клавиатуры:
- Все TextField и TextFormField имеют правильные настройки
- `keyboardType` настроен для каждого поля
- `textInputAction` настроен правильно

Проблема обычно только в настройках симулятора, а не в коде приложения.
